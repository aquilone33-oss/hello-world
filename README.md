# Escape Room Matematica (prototipo)

Questo repository contiene un prototipo web di una escape room matematica per il calcolo letterale.

## Come visualizzare il gioco

Puoi aprire il file `index.html` direttamente nel browser oppure avviare un piccolo server locale.

### Opzione consigliata: server locale

```bash
python -m http.server 8000
```

Poi visita:

```
http://localhost:8000/index.html
```

## Come pubblicare su GitHub

I file creati localmente compaiono su GitHub solo dopo aver effettuato il push sul repository remoto:

```bash
git add .
git commit -m "Add escape room prototype"
git push origin <nome-branch>
```

Sostituisci `<nome-branch>` con il nome del tuo branch corrente.
